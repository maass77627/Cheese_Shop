# Cheese Shop


Cheese Shop

A front-end cheese and charcuterie shop web app for browsing cheeses, cured meats, and related items, with wine pairing recommendations powered by the Spoonacular API.
Built with HTML, CSS, JavaScript, and Bootstrap, using json-server as a mock backend.

How to Run the App
1. Install json-server (if not already installed)
npm install -g json-server

2. Start the backend server

From the project root (where db.json is located):

json-server --watch db.json --port 3000


The API will run at:

http://localhost:3000

3. Open the app in the browser

Open the index.html file in your browser:

Double-click index.html
